An earlier version of this chart read the syllabary numbers as a structural “noise floor.” A convergence control (right-hand panel) shows they were under-converged — the matcher was stopped mid-descent at a fixed 2,000-step budget. Given more compute, Linear B climbs to 37%+ and is still rising. The floor was the clock, not the corpus. The durable lesson survives — and sharpens.

The syllabary floor was a wall clock

A controlled test on scripts we can already read. An unsupervised matcher is handed N words of a known cipher and scored on how much of the true reading it recovers — never shown the key. Small-alphabet scripts converge fast and recover. Syllabaries — the family Linear A belongs to — need far more annealing to converge, and even converged they don’t reach the published SOTA. The lever that closes that gap is not text and not compute: it is an anchor — a known related language.

Syllabaries — under-converged, not floored

The numbers are lower bounds.
  • 2–5% → 37%+Linear B → Greek the closest analog to Linear A. At 2,000 steps it reads ~2–5%; run to 6,000 it climbs to 37.3% and is still rising (Panel B).
  • anchor gap37% → 89% even converged, anchor-free CSA falls short of the published 89%. That last gap needs a pre-grouped cognate lexicon — an anchor.

What it means

For a syllabary the binding constraint is not corpus size and not compute alone. Compute lifts the score a long way (2,000→6,000 steps takes Linear B from ~3% to 37%+), but the remaining distance to a real reading is closed by an anchor — a known related language whose cognates pre-group the search. Linear B has Greek. Linear A has no known relative, so neither more text nor more annealing manufactures the signal that a cognate lexicon supplies.

The lever is the anchor, not the corpus and not the clock.

What it does not mean

Not that Linear A is undecipherable, and not that no method can read it. This is one method (coupled simulated annealing) on analog scripts.

And not what the first version of this chart implied: there is no structural “size wall” — the earlier 2–5% was a stopped clock, a censored lower bound. The honest curve keeps climbing. The alphabet positive controls still hold: the tool demonstrably converges and recovers an easy cipher, so the syllabary shortfall is a property of the problem (anchors), not a broken detector.
