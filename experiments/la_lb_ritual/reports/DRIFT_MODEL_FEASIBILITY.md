# DRIFT_MODEL_FEASIBILITY — §VIII

`channel_class = EXPLORATORY_POSTHOC_CHANNEL`. Generated by `src/drift_feasibility/drift_feasibility.py`.

## Question
Can a drift-tolerant model be specified WITHOUT learning transformations from the known LA↔LB pairs?

## Model families
| family | drift span | available |
|---|---|---|
| D0 exact identity | 0 | ✅ |
| D1 tier-A equivalence | 0 | ✅ (== D0 until allographs clustered) |
| D2 source-grounded uncertainty | ≤1 flagged | ✅ (allograph/damage/composite; NOT vowel drift) |
| **D3 independent constrained edit** | ≥2 subs | ❌ **not independently admissible** |
| D4 permissive edit | unbounded | ✅ (false-positive stress test only) |

## D3 admissibility (all criteria must hold — they do not)
Blocking: `transformations_estimated_from_independent_data = False`;
`weights_cross_validated_on_non_target_data = False`; `target_pairs_absent_from_training = False`.
The vowel-quality substitutions the known drift pairs need (RU→RI = u→i, SA→SO = a→o) have only **WEAK**
independent support; their clear examples **are** the quarantined known pairs, so estimating the model
from them is circular. The derivational suffixes (`-i-jo`, `-de`) are independently attested LB
morphology, but matching a bare LA form to a suffixed LB form is a length change that only a permissive
edit model permits — and permissive affixation inflates false positives.

## Verdict
**A constrained drift model (D3) is NOT independently specifiable.** Only D0/D1 (exact/tier-A) and D2
(≤1 flagged uncertainty) are admissible, and those span **0 genuine drift**. So the ritual channel —
exactly like the closed administrative channel — can test only **exact/near-exact** continuity, not the
drift-inclusive phenomenon. See `INDEPENDENT_DRIFT_EVIDENCE_AUDIT.md`.
