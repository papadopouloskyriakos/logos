## REAL fixes (each fail-closed, each with a regression test)

### P0 — three gate bugs (`scripts/verdict.py`), verified present
1. **MDL clause removed from the gate.** `k_le_u_floor` previously sat inside the AND with a
   free-parameter default, so it could pass degenerate input. Removed as a *clause*; `u_floor` is now
   **reported-only** (NaN-safe) next to every claim (invariant 7 preserved). Regression:
   `test_mdl_clause_removed_u_floor_reported_only`, and `test_gate_clauses_fail_closed_on_degenerate_input`
   asserts `"k_le_u_floor" not in clauses`.
2. **Pre-registered `generalizes_to_virgin` threshold.** The virgin-generalization clause now requires
   a pre-committed `virgin_threshold` AND ≥ `virgin_min_signs` (default 2) distinct supported L_virgin
   signs; it **fails closed** when either is absent (no threshold ⇒ no graduation). Regression:
   `test_litindex_virgin_support_feeds_the_e_gate_via_grade_row`.
3. **Order-statistic bar promoted, DSR demoted.** The operative deflation clause is now
   `beats_order_stat_bar` (observed must exceed E[max] of the null over `n_trials`), which needs no
   normal approximation and fails closed on a zero-variance null. DSR is still computed/reported.
   `n_eff`→`n_trials` (and `n_eff_source`→`n_trials_source`) throughout. Regression: degenerate constant
   null ⇒ `beats_order_stat_bar` False.

### P1 — null-simulation robustness
4. **`n_fake` default 16 → 300** (`run_canary.py` `run()` + CLI), so the corrected-margin bar (Cornish–
   Fisher on the L_fake skew/kurtosis) is estimated from enough instances to resolve higher moments;
   `nulls.null_distribution` draws `n_packard=n_random=n_within` 32 → 100.
5. **Exact rank p-values** in `morphostat.s_morph` (`n_null` 64 → 1000): the decision is now
   `p_rank = (#{null ≥ score}+1)/(B+1)` vs a Šidák/normal α, replacing the normal-approx z-test; adds
   `p_rank`, `p_rank_se`, `alpha`, `n_null`. Note: `s_morph` is called **only** by `verdict.py` (the
   gate's morph clause) — not by the `morphology.py` probe — so this hardens gate decisions, not the
   published morphology numbers.

### P2 — cross-script image result: font-swap control + honest relabel
6. **Font-swap confound control** (`scripts/palaeo/font_control.py`, artifact
   `results/palaeo/font_control.json`; `render_glyph` now takes a `font_path`). The baseline draws BOTH
   scripts in one face (Aegean); the control re-renders Linear A in **Noto Sans Linear A** and Linear B
   in **Noto Sans Linear B** — two independent single-script designers (cmap-verified: NotoA has 0
   Linear B glyphs, NotoB 0 Linear A). **Result: the font-artifact hypothesis is REFUTED** — recovery
   survives and slightly strengthens (direct-NN 0.367 → 0.416; Procrustes 0.156 → 0.195; chance 0.0135;
   ~113% retained). The image leg is nonetheless **relabelled a "shape-genealogy engineering
   demonstration, not an archaeological positive"** for the deeper, un-controllable reason the font swap
   cannot touch: the A↔B anchor *values* were assigned by epigraphers on the basis of sign shape (Linear
   B descends graphically from Linear A), so recovering that value map from glyph images is largely
   circular. I-JEPA is kept out of the main claims (over-parameterized for ~90 glyphs). Regression:
   `tests/test_font_control.py` (3 tests, cmap single-script property + mechanical verdict + determinism).
   The preprint's "one modest offensive positive" framing is withdrawn (§7.3, §8, §9): **the study
   reports no offensive positive.**
