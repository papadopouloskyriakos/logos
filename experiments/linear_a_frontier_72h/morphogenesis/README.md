# TURING_MORPHOGENESIS — graph reaction–diffusion for latent linguistic structure (E091–E096)

A priority frontier family inside the LOGOS 72h Linear A campaign. Tests whether a graph reaction–diffusion (Turing)
system recovers **known** linguistic structure in **blinded** Linear B, whether that is **Turing-specific** (vs
generic graph clustering), whether the signal **survives degradation** to Linear-A-like conditions, and — only if the
calibration + specificity gates pass — applies a **frozen** model to Linear A at the **L2/L3 ceiling** (anonymous
classes / segmentation probabilities / regional classes; **never** sound values, translations, or "pattern=phoneme").

- `PREREGISTRATION.md` — frozen commitments for all six epochs.
- `THEORY_AND_PRIOR_ART.md` — the mechanism, the graph analogue, and why generic clustering ≠ morphogenesis.
- `GRAPH_SCHEMAS.md` / `PARAMETER_SPACE.md` / `SEARCH_RECEIPT.md` — graph views, searched space, multiplicity ledger.
- `scripts/` harness (graphs, rd, evaluate); `controls/`, `results/`, `reports/` per epoch.

Coordinator = Claude (build + verify + bank); heavy repetitive nulls may be offloaded. The 74-node LB graph runs
locally in seconds. Every epoch: verify Turing conditions → positive control → negative + adaptive nulls →
mechanical verdict → persist → successor. **A failed calibration is a valid epoch.** Finalization stays gated.
