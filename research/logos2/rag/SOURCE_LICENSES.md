# E211-RAG source licences

Per-source licence posture is in SOURCE_REGISTER.csv. Hard rules: copyrighted
chapters index METADATA/citations only (no full text); Younger pages internal-use factual
extraction, no redistribution; licensed raw vendor data (GORILA/SigLA/DAMOS raw) never
indexed as text, only derived records already committed in-repo; every blocked source
recorded. Index D content is claims-only and never support. The review bundle carries
hashes + acquisition instructions for non-redistributable sources, never their text.
