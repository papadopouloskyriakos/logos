## The decision this unblocks: is the frozen morphology pre-reg testing a stale target? — NO

**§7 "What Do We Know from Reading Linear A?" confirms the Davis 2013 inventory the pre-reg
encodes.** Salgarella's "reading+ approach" (read structurally, decline a literal "translation")
endorses:
- **`i-*301` = "(s/he) gives/dedicates"** — the stable verbal root of the libation formula, variably
  affixed (Davis 2013, earlier Duhoux 1992), reported as compelling. This is the richest single field
  confirmation of registered **H1a**, and it is a *structural/positional* reading: **`*301` is left
  undeciphered (no phonetic value assigned)** — exactly logos's no-phonetic-claim posture.
- **VSO word order (H1b)** — but hedged: "Minoan **may be** a Verb-Subject-Object (VSO) language"
  (Davis 2013), explicitly "not enough … to claim an affiliation." (Calibration: not "compelling.")
- The six-position libation-formula template (Davis 2013): 1 main verb / 2 place-name / 3 dedicant /
  4 `A/JA-SA-SA-RA-ME` object "dedication/offering" / 5 subordinate verb / 6 `-TE` "from" phrase.

**Thomas 2020 is NOT cited.** Salgarella segments **whole prefixal units `A-TA-` / `TA-NA-`** (Davis),
*not* Thomas's `a-/ta-/na-/t-` morpheme cuts. So Thomas's segmentation is a **competing** analysis,
neither confirmed nor adopted — it must be registered **head-to-head** against the Davis whole-unit
analysis, not silently assumed. → new pre-reg addendum (below).

**Consequence:** the frozen `prereg-morphology-2026-06-30.md` is **not stale** (its Davis H1 target is
endorsed by the current synthesis). The needed action is to **add**, via a new dated addendum, the
structures Salgarella supplies that the frozen pre-reg does not encode.

## §8 "Which Language Does Linear A Encode?" — the language benchmark (all claims verified)

- **Verdict: Minoan is a language ISOLATE** — "does not belong in any known language family" (the
  families "used for comparative purposes being Indo-European, Semitic, Afro-Asiatic"); "an
  'isolated' language, indigenous to Crete" (via Davis 2014, Duhoux 2020). **Typology:** agglutinative
  (multisyllabic prefixing; Duhoux 1978) + provisional VSO (Davis 2013).
- **`*301` is never mentioned in §8; Di Mino is never cited** (grep-verified: zero hits). So **§8 does
  not, by itself, resolve the logos "pending" item** — it does not adjudicate `*301`. The Semitic
  lineage is named only via **Gordon (1966/1969) and Best (1972/2001)** and dismissed at the **method**
  level: the etymological method "is fraught with problems"; "vocabulary alone is not probative …
  words can easily be borrowed." This maps directly onto **logos invariant 8** (the "English is a
  Semitic language" cherry-picking failure mode), *not* onto a sign-by-sign rebuttal of `*301`.
- **Method ranking — logos's lane named explicitly.** Combinatorial/context method "more promising
  than the etymological method"; Davis's syllabotactic analysis "at the forefront"; **digital/
  statistical analysis is "the next desideratum in the field."** logos can position itself as exactly
  that desideratum, while respecting Duhoux's 11-step guardrails (GORILA-edited texts only, systematic
  cross-text patterns, no anachronism).
- **Benchmark-blessed affix inventory** (held-out targets for `verdict.py`, with attributions for
  stratification): `-TE/-TI` = "from"/"of" (Valério 2007); `I-/J-` = "to"/"at" (Duhoux 1997); `-JA` =
  adjective-forming (Younger 2024); **`-RU/-RE` ~ Greek masculine `-os`** (Steele & Meissner 2017, via
  personal-name comparison). Note `-RU/-RE` is the v2-flagged candidate case-marker, now sourced.
- Corpus envelope: "ca. 2,500 inscriptions"; **"no bilingual text has so far been found."**

## §4 (palaeography/form), §6 (genres), §1–3 (corpus baseline)

- **§4 — the A vs AB split scopes the image claim.** `A`-prefix = Linear-A-only; `AB`-prefix = shared
  homomorph (phonetic value borrowed from Linear B). **`*301` is an A-only sign**, so an image match on
  it carries *shape*, not transferred phonetic value — the cross-script image "offensive signal" must
  be labelled by the A/AB split. GORILA V = **390 signs** (~150 syllabograms / ~200 ideograms / <50
  fraction signs "klasmatograms"; numerals + the one punctuation dot not in the 390). **Form is not
  diagnostic of function** — so any glyph-shape→role classifier (JEPA image) is a ≤0.75 truth-layer
  signal, never a decider (invariant 5). `A559 = MA+RU` is the one validated monogram-reads-as-word
  landmark.
- **§6 — the stratification strata, concrete at last.** Clay ≈ **90%** of the corpus (primarily
  administrative: tablets + sealings); non-clay ≈ **10%** (mostly cultic/dedicatory; stone-vessel
  libation formula ~50 inscribed objects at peak sanctuaries/caves). Clean three-way strata for the
  stratified re-run: **HT/palatial admin tablets vs libation/votive stone-vessel vs heterogeneous
  other**, with GORILA doc-type codes (Za stone/libation, Wa/Wb/Wc sealings, Zb clay vessels, Zg
  figurines, Zd graffiti, Ze architectural, Zf metal) as stratum labels. Commodity ideograms (AB120
  GRA, AB122 OLIV, AB131 VIN, …) must be **masked** before any phonetic-morphology pass on the admin
  stratum.
- **§1–3 — count reconciliation (invariant 12).** The sign-count confusion resolves into **four
  different denominators**, each to be labelled where used: **97** (Braović/Tan token-frequent
  syllabograms) / **~150** (full syllabary estimate) / **259** (logos working repertoire) / **~350**
  (total incl. ideograms). Document/corpus: **~1,400 readable** inscriptions of **~2,500 total** finds,
  **~7,400 sign occurrences**, **56 findplaces**, ca. 1800–1450 BCE (MM II–LM IB), overwhelmingly the
  single **LM IB** destruction horizon. **The information-floor denominator is ~7,400 occurrences, not
  1,400 docs** (invariant 7). The near-single horizon **corroborates the stratification addendum's
  "decline chronological CV"** — a site-held-out split is the only realistic partition.
