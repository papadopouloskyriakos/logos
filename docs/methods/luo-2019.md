**"Neural Decipherment via Minimum-Cost Flow: from Ugaritic to Linear B"** — ACL 2019.
[arXiv 1906.06718](https://arxiv.org/abs/1906.06718) · [ACL P19-1303](https://aclanthology.org/P19-1303.pdf).

This is the seminal automatic-decipherment result and **logos's methodological baseline** —
the thing we reimplement and extend, not reinvent. (No open code is released → we reimplement
from the paper.) It is *both* the proof-of-concept for the whole approach *and* the
proof-of-limit for Linear A.

## The method (one paragraph)

Frame decipherment as recovering a character-level mapping from a **ciphertext** (the lost
script) to a **plaintext** (a *known*, related language). An expressive **sequence-to-sequence
model** captures correspondences between cognates; to train it without supervision, the
training is formalized as a **minimum-cost flow** optimization (a constrained assignment that
finds a globally-consistent character mapping). The model is **informed by patterns in
language change** documented in historical linguistics (regular sound correspondences). The
decisive precondition: **you need a known cognate language to map *to*.**

## The results (the honest, load-bearing ones)

- **Ugaritic → Hebrew:** +5.5% absolute over prior SOTA. A full auto-decipherment that
  *worked* — because Ugaritic is a Semitic language closely related to Hebrew. (Ugaritic is,
  not coincidentally, one of the 8 languages Di Mino reads.)
- **Linear B → Greek:** **67.3% of cognates correctly translated** — the first automatic
  result on Linear B. Linear B's language *is* known (Mycenaean Greek), so a cognate target
  exists; the method gets most of the way.
- **Linear A: not addressed — and would fail by construction.** Linear A's language is
  **unknown** and has no accepted living/known cognate. With no plaintext to map to, the
  minimum-cost-flow objective has nothing to optimize toward. **This null is the empirical
  version of logos's unicity-distance argument** (`scripts/corpus_info.py`): the information
  is not in the data, and no method manufactures it.

## What logos reimplements (the reproducible baseline)

1. **Ugaritic→Hebrew success** — proves our pipeline + a faithful port of the method.
2. **Linear-B partial (target ~67%)** — a second validation against a known-language control.
3. **The Linear-A null** — run the same machinery on Linear A and *show it cannot pin a
   map* without a cognate. This is the honest baseline milestone: logos's discipline
   machinery must reproduce the field's own floor before it tries to beat it.

## What logos adds on top of Luo (the differentiation)

Luo is **pure optimization** — no LLM, no eval flywheel, no multiple-testing discipline, no
held-out acceptance gate. logos's extensions:

- **The agora discipline layer** — Deflated-Sharpe / effective-n over (signs × roots ×
  families tried): the cure for the "English is a Semitic language" cherry-picking that pure
  cognate-matching invites. Luo has no equivalent.
- **Agentic LLM hypothesis generation** — Claude proposes candidate cognate mappings / sound
  laws as **≤0.75 signals** (truth-capped; Luo has no proposer). The LLM never grades itself.
- **Mechanical held-out verdicts** — a reading must read held-out sites/inscriptions; the
  Libation Formula × 5 sites is a natural 5-fold CV. Luo reports corpus-wide accuracy, not
  held-out acceptance.
- **Cross-script A↔B JEPA transfer** — a learned joint embedding that *infers* a cognate-like
  structure from the 60 shared signs, partially supplying the "plaintext target" Luo's method
  requires but Linear A lacks. (This is the one genuinely novel bet; gated behind the
  baseline.)
- **Open platform** — code, verdict methodology, lexicon tables public. Luo released none.

## Honest placement

logos's defensible scholarly claim is **"Luo 2019 + the agora honesty layer + agentic
generation + cross-script JEPA transfer, applied openly to Linear A"** — an extension of a
peer-reviewed method, not "another amateur cracked Linear A." If the cross-script JEPA can
manufacture a usable cognate target where none exists, that is the real contribution; if it
cannot, the honest deliverable is a rigorous, deflation-correct map of what the 7,500-char
corpus can and cannot support. Both are valuable; only one is a decipherment.
