## Per-candidate expected gain (ranked)

Notation: **ΔConstraints** (new independent constraints), **ΔEqClasses** (equivalence classes broken),
**ΔPower** (ordinal, on effective_n / falsification), **Layer**, **Independence** (of GORILA→Ventris).

### 1 — Anetaki II (KN Zg 57/58 edition) — **HIGH**
- **ΔConstraints:** one ~119-sign continuous text ≈ **3×** the current longest; ~+2% to the in-word sign mass
  (5,792→~5,911) but concentrated as *long-range within-text context* — the exact quantity the substitution
  channel is starved of. Enables position-in-long-run C/V contrasts impossible on 1.84-sign words.
- **ΔEqClasses:** collapses positional C/V degeneracies (fixed positions in a long run); resolves
  segmentation ambiguities in a monolingual continuous text; possibly a few sign-ID calls via fresh autopsy.
- **ΔPower:** the decisive gain is **prospective falsification power**, not effective_n: it grades every
  pre-frozen Anetaki prediction (Chiapello 2024 archived, Rjabchikov 2025 / Di Mino 2026 quarantined) with
  zero circularity (Art. XII). This is the Linear-B-new-tablet standard (Invariant 3).
- **Layer:** L2–L4-context, L7. **Independence:** YES (archaeology, non-value lineage).
- **Honest cap:** monolingual → authorizes **no** SEMANTIC+ licence by itself. It can *refute* candidates; it
  cannot *confirm* values without an independent second channel.

### 2 — New LA inscriptions + joins — **MED-HIGH**
- **ΔConstraints:** each find extends held-out supply; **joins** merge fragments → longer sequences (the
  scarce long-context resource) and remove artificial word breaks at fracture points.
- **ΔEqClasses:** joins collapse boundary ambiguity; new sites break the site-clustered confound that the
  segmentation gap CI had to control for.
- **ΔPower:** cumulative effective_n growth; the only *organic* route to more held-out gold. Individually
  small (mean 1.84 words).
- **Layer:** L1–L3. **Independence:** YES.

### 3 — Sign catalogues / rationalized sign-lists — **MED**
- **ΔConstraints:** none textual — **fixes the denominator N** (inventory size) that every significance figure
  divides by (Art. IX / Invariant 7). A wrong N silently inflates or deflates deflated-significance.
- **ΔEqClasses:** merges/splits sign classes → directly changes effective_n and the unicity budget.
- **ΔPower:** *hygiene* gain (prevents false graduation), not new signal. **Layer:** L1. **Independence:** YES
  (palaeographic).

### 4 — Re-editions / GORILA corrigenda — **MED**
- **ΔConstraints:** resolves dotted/uncertain signs → fewer masked positions in the substrate.
- **ΔEqClasses:** splits/merges allographs; lowers L1 label noise that inflates the null.
- **ΔPower:** raises the *ceiling* of every downstream test by cleaning inputs. **Layer:** L1.
  **Independence:** NO (GORILA lineage) → cannot serve as an independent anchor, only as substrate cleanup.

### 5 — SigLA version bumps — **MED**
- **ΔConstraints/EqClasses:** new hand drawings / bounding boxes; refines allograph classes; a candidate
  **second independent (shape) channel** for the ≥2-channel seal requirement (WP-G/J).
- **ΔPower:** incremental; already-exploited channel. **Layer:** L1–L2. **Independence:** YES (shape≠sound).

### 6 — Cretan Hieroglyphic — **MED-LOW**
- Independent (undeciphered → uncontaminated) *structural comparandum*, one script removed. Supports
  cross-script channel design; decides no LA layer. **Layer:** L2 / L6-comparandum. **Independence:** YES.

### 7 — Museum image / autopsy releases — **LOW-MED**
- Independent visual channel; can turn a masked/damaged sign into an observed one (L0→L1). High value **only**
  when paired with a new/long inscription (e.g. scepter plates); marginal alone. **Independence:** YES.

### 8 — Cypro-Minoan updates — **LOW**
- One script removed, undeciphered, structural-only → detector control, not an LA constraint source.

### 9 — Onomastic / toponym indexes — **LOW (capped, Art. XII)**
- Every toponym anchor rides on GORILA homomorphic values. Grading a value hypothesis with GORILA-derived
  toponyms is **circular** — kept for `litindex` decontamination only, never scored as evidence. Now
  archived-only (Kansas host withdrawn).
